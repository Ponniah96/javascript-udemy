JavaScript Fundamentals – Part 1

Values and Variables

Variables

variables are used to store data values in JavaScript.

naming convention while calling variables

  • cannot start with number
  • cannot contain space or hyphen (-)
  • cannot be a reserved keyword (new, class, etc.)
  • use camelCase for multiple words

Primitive Data Types

  • Number: integer and decimal numbers (e.g., 23, 23.5)
  • String: sequence of characters enclosed in quotes (e.g., 'Hello')
  • Boolean: true or false values
  • Undefined: variable declared but not assigned a value
  • Null: represents an intentional absence of any object value
  • Symbol: unique and immutable data type used as identifiers for object properties
  • BigInt: represents integers with arbitrary precision (e.g., 9007199254741991n)

Dynamic Typing

JavaScript is a dynamically typed language, meaning you don't have to specify the data type of a variable when you declare it. The type is determined automatically based on the assigned value and can change at runtime


typeof Operator

The typeof operator is used to determine the data type of a variable or value in

Basics of let, var, const

  • let: used to declare block-scoped variables that can be reassigned.
  • const: used to declare block-scoped variables that cannot be reassigned after their initial assignment.
  • var: function-scoped variable declaration (older way, less preferred now).

Basic Operators

  • Arithmetic Operators: +, -, *, /, %, **
  • Assignment Operators: =, +=, -=, *=, /=
  • Comparison Operators: ==, ===, !=, !==, >, <, >=, <=
  • Logical Operators: && (AND), || (OR), ! (NOT)
  • Increment and Decrement Operators: ++, --

Operator Precedence

Operator precedence determines the order in which operators are evaluated in an expression. Operators with higher precedence are evaluated before those with lower precedence. For example, multiplication (*) has higher precedence than addition (+), so in the expression 3 + 5 * 2, the multiplication is performed first, resulting in 3 + 10 = 13.

  • Parentheses: ()
  • Exponentiation: **
  • Multiplication, Division, Modulus: *, /, %
  • Addition, Subtraction: +, -
  • Comparison Operators: <, <=, >, >=
  • Equality Operators: ==, ===, !=, !==
  • Logical AND: &&
  • Logical OR: ||

Strings and Template Literals

Strings are sequences of characters used to represent text in JavaScript. They can be created using single quotes (' '), double quotes (" "), or backticks (` `). Template literals, enclosed in backticks, allow for multi-line strings and embedding expressions using ${expression} syntax.

Template Literals

Template literals are a way to create strings in JavaScript that allow for easier embedding of variables and expressions. They are enclosed in backticks (` `) instead of single or double quotes. Template literals support multi-line strings and expression interpolation using the ${} syntax.

Advantages of template literals

  • Multi-line strings without the need for escape characters.
  • Embedding variables and expressions directly within the string.
  • Improved readability and maintainability of code.

if else statements

The if...else statement is a conditional statement that allows you to execute different blocks of code based on whether a specified condition is true or false. If the condition evaluates to true, the code block within the if statement is executed. If the condition is false, the code block within the else statement (if provided) is executed instead.

Type Conversion and Coercion

Type conversion is the process of converting a value from one data type to another

Type Coercion

Type coercion is the automatic or implicit conversion of values from one data type to another by JavaScript. This occurs when performing operations involving different data types, such as combining a string and a number.

Truthy and falsy Values

In JavaScript, truthy and falsy values are used in conditional statements to determine the flow of execution. A truthy value is any value that is considered true when evaluated in a boolean context, while a falsy value is any value that is considered false.

Falsy values in JavaScript include:

  • false
  • 0 (zero)
  • "" (empty string)
  • null
  • undefined

Equality Operators: == vs ===

The == operator performs type coercion before comparing values, while the === operator checks for both value and type equality without coercion.

It's generally recommended to use the === operator to avoid unexpected type coercion and ensure strict equality checks.

Prompt

The prompt() function is used to display a dialog box that prompts the user for input. It takes an optional string argument that serves as the message displayed in the dialog box. The function returns the user's input as a string.

Example:

const userInput = prompt("Please enter your name:");
console.log(userInput);

Logical Operators

Logical operators are used to combine multiple conditions or boolean values in JavaScript. The three main logical operators are AND (&&), OR (||), and NOT (!).

  • AND (&&): Returns true if both operands are true.
  • OR (||): Returns true if at least one operand is true.
  • NOT (!): Returns the opposite boolean value of the operand.

Switch statements

The switch statement is a control flow statement that allows you to execute different blocks of code based on the value of a variable or expression. It is often used as an alternative to multiple if...else statements when you have several possible values to check against.

Statements and expressions

expression

An expression is any valid unit of code that resolves to a value. It can be as simple as a single value or variable, or it can be a more complex combination of values, variables, and operators that produces a new value.

statement

A statement is a complete unit of code that performs an action. Statements are made up of one or more expressions and can include control flow structures, variable declarations, function definitions, and more. Statements do not produce a value themselves but rather execute a specific task.

Ternary operators

The ternary operator is a concise way to write conditional statements in JavaScript. It is represented by the ? and : symbols and takes three operands: a condition, an expression to execute if the condition is true, and an expression to execute if the condition is false.

Syntax:

condition ? expressionIfTrue : expressionIfFalse;

Advantages:

  • Concise syntax for simple conditional statements.
  • Can be used within expressions, making it useful for inline conditions.
