User Snippets

Go to Code > Preferences > User Snippets > New Global Snippets file > name it as javascript.json > add below code 

{
  "Print to console": {
    "prefix": "log",
    "body": ["console.log('$1');", "$2"],
    "description": "Log output to console"
  }
}

Debugging Tips

  • Use console.log() to check variable values
  • Use breakpoints in developer tools
  • Check error messages carefully
  • Isolate the problem area
  • Review recent code changes

Different types of console methods

  1. console.log(): General output of logging information.
  2. console.warn(): Outputs a warning message.
  3. console.error(): Outputs an error message.
  4. console.table(): Displays data as a table.
  5. console.debug(): Outputs a debug message.
  6. console.trace(): Outputs a trace message.
  7. console.time() and console.timeEnd(): Measures time taken by a block of code.
